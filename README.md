# 如何开始
 - 克隆本项目到本地
    `git clone git@github.com:FeyeW/musicapp.git`
 - 后台API地址（再次感谢强大后端开源API🎉）
    [GitHub指路🙋](https://github.com/Binaryify/NeteaseCloudMusicApi)
  
 - 运行
    `npm run dev`
## 项目简介
- 项目结构
```
├─ src
    ├─api                   // 网路请求代码
    ├─assets                // 字体配置及全局样式
    ├─style                 // 公共样式
    ├─components            // 可复用的 UI 组件
    ├─utils                 // 工具类函数和相关配置
    ├─views                 // 页面
    ├─router                // 路由配置文件
    └─store                 // redux 相关文件
      App.jsx               // 根组件
      main.jsx              // 入口文件
```
